# 风格原型库

零素材探索模式的冷启动基底。每个原型是一套完整、自洽的 token 预设——用户不是从零选，是从成熟方案里挑了再改。

用法：**速筛轮**把全部 16 个原型渲染成实景卡供三态投票（协议见 `references/picker-protocol.md`《速筛轮》）——心动票圈定结构来源，排除票进红线池；下面的《映射建议》只决定卡片排序，不决定谁被隐藏。**方向板轮**以票果圈定的原型为组合原料——版式签名做结构层（构图语法 + 明度结构 + 密度）、色板字体做初始样式层，输出 3-4 个结构级差异的**组合与变体**（禁止原样复读单一原型，生成纪律见 SKILL.md ③），并各自在 2-3 个画幅场景上实例化（见 `picker-protocol.md`《方向板轮》）；**细节轮**再从各原型取材出区块候选。**原型值只是起点，循环微调才出个人化**；候选生成必须遵守文末《混搭规则》，候选不得整族集中。

气质族用于读票：同族多卡被排除 → 整族降权，方向板不再从该族取结构。

## 气质族总览

| 族 | 原型 | 共同气质 |
|----|------|---------|
| 纸感工坊 | 复古印刷 · 手账涂鸦 · 复古旅贴 | 暖纸底、手作温度、实物感 |
| 理性编辑 | 杂志编辑 · 极简瑞士 · 包豪斯几何 | 留白、网格、克制或几何 |
| 书卷东方 | 学院羊皮 · 水墨宣纸 | 沉静、文人气、时间感 |
| 暗色夜场 | 暗黑终端 · 夜读烛光 · 像素街机 · 青图纸 | 深色底、夜间或档案感 |
| 活力撞色 | 新粗野 · 孟菲斯糖果 | 高饱和、生猛、年轻 |
| 柔和记录 | 柔和粉彩 · 胶片暗房 | 低攻击性、记录感、温柔 |

## 1. 复古印刷 Retro Print

- **气质**：新闻纸、油墨、活字。粗粝、复古、有报刊权威感
- **版式签名**：报版式——通栏大标题压顶、双细线分栏、首字下沉；油墨黑对亮纸高对比；密度高、铅字紧排
- **适合**：评论观点类、历史/文化内容、独立刊物感
- **色板**：底 `#F5F1E8` 新闻纸 / 文字 `#1A1A1A` 油墨黑 / 点缀 `#C0392B` 印刷红 / 次级 `#8B8378` 灰褐
- **字体**：粗壮老式衬线（Bitter / Source Serif 4）× 中文宋体（Noto Serif SC 900）
- **质感**：半调网点、纸张噪点
- **元件**：双细线框（票据感）、首字下沉、报头横线、"第 N 期"编号
- **红线种子**：忌渐变、忌圆角、忌高饱和荧光色

## 2. 杂志编辑 Editorial

- **气质**：大标题、大留白、克制的一抹点缀色。高级、安静、内容为王
- **版式签名**：大留白——超大标题占上 1/3、全画面单焦点；明度对比全集中在标题区；密度低
- **适合**：深度长文、访谈、生活方式、知识付费
- **色板**：底 `#FFFFFF`→`#FAFAF8` / 文字 `#111111` / 点缀单色（如 `#D4380D` 橘红或 `#0958D9` 克莱因蓝，二选一）/ 次级 `#999999`
- **字体**：超大号现代衬线标题（Libre Caslon / Lora）× 无衬线正文（Noto Sans SC）
- **质感**：无纹理，靠留白和字号层级
- **元件**：极细分隔线（0.5-1px）、页码/栏目角标、引文超大引号
- **红线种子**：忌元素拥挤、忌多于一个点缀色、忌描边字

## 3. 极简瑞士 Swiss Grid

- **气质**：网格、对齐、功能主义。理性、现代、信息密度可以很高
- **版式签名**：网格驱动——左对齐一切、色块分区、超大序号做锚点；明度均匀、信号红点睛；密度可高但必须严格对齐
- **适合**：数据/图表向内容、产品测评、效率工具教学
- **色板**：底 `#F4F4F4` / 文字 `#0A0A0A` / 点缀 `#E30613` 信号红 / 辅助 `#B5B5B5`
- **字体**：Grotesk 无衬线全家桶（Archivo / Space Grotesk）× 中文黑体（Noto Sans SC）
- **质感**：无纹理；可见网格线作为设计元素
- **元件**：色块分区、超大序号、箭头符号 →、左对齐一切
- **红线种子**：忌衬线、忌装饰性元素、忌居中对齐

## 4. 暗黑终端 Terminal

- **气质**：近黑底、荧光单色、等宽字。极客、夜间、代码原生
- **版式签名**：终端窗口——顶部状态栏、内容左对齐逐行排、`$` 行首；暗底亮字、亮区小而散；密度中
- **适合**：编程教学、命令行工具、黑客向内容
- **色板**：底 `#0D1117` / 文字 `#C9D1D9` / 点缀 `#3FB950` 终端绿（或 `#E3B341` 琥珀）/ 结构 `#30363D`
- **字体**：等宽为主（JetBrains Mono / IBM Plex Mono）× 中文黑体（Noto Sans SC）
- **质感**：细扫描线或微噪点（可选）
- **元件**：`$` 提示符、闪烁光标块、ASCII 框线、状态栏
- **红线种子**：忌彩虹语法高亮配色当品牌色、忌亮底、忌衬线

## 5. 手账涂鸦 Notebook

- **气质**：横线本、马克笔、胶带贴纸。亲切、随性、人味儿重
- **版式签名**：散点拼贴——元素带手感偏移、便利贴/胶带随性锚定、无严格网格；亮底中对比；密度中高但松弛
- **适合**：个人 vlog 式分享、学习笔记、生活技巧
- **色板**：底 `#FFFDF5` 奶油 / 文字 `#3D3A35` 深暖灰 / 点缀多彩低饱和（`#F4A261` 橘 `#2A9D8F` 青 `#E76F51` 红）
- **字体**：手写体标题（LXGW WenKai / Ma Shan Zheng）× 圆润正文（Noto Sans SC）
- **质感**：横线纸 / 点阵纸
- **元件**：荧光笔划线、和纸胶带、手绘箭头圈圈、便利贴卡片
- **红线种子**：忌精确对齐（要有手感偏移）、忌企业感图标、忌深色底

## 6. 柔和粉彩 Soft Pastel

- **气质**：低饱和粉彩、大圆角、柔光。温柔、治愈、零攻击性
- **版式签名**：居中柔焦——圆角卡片居中浮置、四周匀称留白；亮底低对比；密度低
- **适合**：女性向、亲子、心理/健康、美学生活
- **色板**：底 `#FBF7F4` 奶白 / 文字 `#5C5470` 灰紫 / 点缀 `#F2BAC9` 粉 `#B8D8D8` 雾蓝 `#F7E3AF` 鹅黄
- **字体**：人文无衬线（Quicksand / Varela Round）× 中文圆体气质（Noto Sans SC 300-500）
- **质感**：无纹理或极淡渐变
- **元件**：大圆角卡片（16px+）、柔光阴影、圆点装饰、丝带标签
- **红线种子**：忌高对比黑、忌直角、忌密集排版

## 7. 包豪斯几何 Bauhaus

- **气质**：三原色 + 黑、几何模块、海报感。直接、现代、有冲击但不闹
- **版式签名**：海报式——对角线/大几何块分割画面、超大标题即构图主体；色块间明度对比强烈；密度低中、每块干净
- **适合**：设计/建筑/艺术向内容、视觉作品集
- **色板**：底 `#F2EFE9` 暖灰白 / 文字与结构 `#1B1B1B` / 点缀 `#BE3A34` 红 + `#27539B` 蓝 + `#E3A92B` 黄（一主二辅）
- **字体**：几何无衬线标题（Archivo Black）× 中文黑体（Noto Sans SC）× 等宽（Space Mono）
- **质感**：无纹理，大色块分区
- **元件**：圆/三角/方几何符号、对角线分割、粗下划线、超大几何序号
- **红线种子**：忌柔光投影、忌粉彩、忌手写体

## 8. 学院羊皮 Academia

- **气质**：羊皮纸、墨绿、烫金。书卷、沉静、老图书馆
- **版式签名**：书页式——居中对称、双线框包页、章节号开篇；中明度均匀；密度中、行距从容
- **适合**：读书博主、人文历史、深度知识付费
- **色板**：底 `#F3EDE0` 羊皮纸 / 文字 `#2C3A2F` 墨绿 / 点缀 `#8C6F3E` 烫金 / 辅 `#6E4F3A` 皮革棕
- **字体**：古典衬线标题（Cormorant Garamond / EB Garamond）× 中文宋体（Noto Serif SC）× 等宽克制使用
- **质感**：细纸纹噪点
- **元件**：双细线烫金框、罗马数字章节号、藏书票徽章、首字下沉
- **红线种子**：忌高饱和、忌无衬线大标题、忌大圆角卡片

## 9. 胶片暗房 Film Lab

- **气质**：相纸、颗粒、日期戳。记录感、有温度的时间味
- **版式签名**：相册式——照片框做画面主体、文字信息缩进角落（日期戳位）；明度跟照片走、框外干净；密度低

- **字体**：中性无衬线（Archivo）× 中文黑体（Noto Sans SC）× 日期戳用 VT323
- **质感**：胶片颗粒噪点
- **元件**：宝丽来白边框、右下角橙色日期戳、胶片齿孔条、帧号标注
- **红线种子**：忌冷调纯白、忌花哨装饰、忌大衬线标题

## 10. 新粗野 Neo-Brutalism

- **气质**：粗黑描边、硬投影、高饱和色块。生猛、直接、反精致

## 混搭规则

原型是预设不是套餐。**版式签名是方向板的结构层**——方向板轮整板单选锁定结构层之后，细节轮的五个区块（色板/字体/质感/元件/形状）是**独立轴**，任何区块都可以从不同原型取材——这是个性的主要来源。结构层不仅可以混，方向板轮**必须**输出组合与变体（生成纪律见 SKILL.md ③）："杂志的构图 × 终端的明度结构"这种跨原型组合是方向板的常态，原样复读单一原型是规格违例。

候选生成纪律：

- **候选不得整族集中**：第一轮每个区块的候选至少 1 个来自不同原型（色板出了羊皮系，元件区块就混入印刷的票据框、手账的胶带）
- **方向级差异，不是微调级**：同区块候选之间至少一项显著不同（底色明度/色温、点缀色相、字体类别）——三个肉眼难分的暖白底不是三个候选，是一个候选浪费了三个名额。微调级差异（同方向 ± 一步）交给 adjuster 档位，不占候选名额
- **七三配比**：第一轮候选七成贴访谈/素材画像（保命中率），三成试探边界（用户没说想要、但也没排除的方向）——访谈负责提高命中率，收敛是循环的工作，不要在第一轮替用户提前收敛
- **原值只是锚点**：候选 = 原型原值 + 按访谈微调的变体，永远给"原型 ± 一步"的选择空间
- **第二轮起跟着用户走**：用户锁定的区块暴露了真实口味，重出候选时以已锁区块的气质为锚跨原型取材

配方示例（气质上自洽的组合，可直接做候选）：

- 学院羊皮色板 × 复古印刷元件 = 古籍刊物
- 水墨宣纸色板 × 极简瑞士排版 = 现代东方
- 柔和粉彩色板 × 新粗野描边 = 软糖贴纸感
- 暗黑终端底 × 手账涂鸦元件 = 极客手帐
- 胶片暗房元件 × 复古旅贴色板 = 旅行底片集
- 复古印刷色板 × 暗黑终端等宽字 = 活字代码

冲突警告（不是禁止，是需要刻意处理才成立）：

- 暗色底 × 纸纹理 → 纹理须换亮色低透明度，否则看不见
- 手写标题 × 像素/瑞士系 → 气质打架，需要其一让位
- 大圆角柔光 × 粗黑描边 → 精致与生猛互斥，二选一
- 点缀色超过 2 个 × 任何"克制"系色板 → 就不再克制了
