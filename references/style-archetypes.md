# 风格原型库

零素材探索模式的冷启动基底。每个原型是一套完整、自洽的 token 预设——用户不是从零选，是从成熟方案里挑了再改。

用法：把开场访谈的答案（领域/受众/气质词/最讨厌的风格）映射到 2-3 个原型，每个原型再按访谈微调出 1 个变体，组成第一轮 picker 候选。**原型值只是起点，循环微调才出个人化**。

反向题选项就用这 7 个原型的名字——用户说"最讨厌哪个"直接排除一族。

---

## 1. 工程蓝图 Engineering Blueprint

- **气质**：暖纸 + 冷蓝图 + 金属点缀。手作感与工程精度并存，克制可信
- **适合**：独立开发者、build in public、硬核技术拆解
- **色板**：底 `#F9F3E5` 暖奶油 / 次底 `#F2EBDB` / 文字 `#465670` 墨蓝灰 / 结构线 `#2B3358` 靛蓝 / 点缀 `#BD7D36` 古铜金
- **字体**：标题衬线（Fraunces）× 等宽标注（IBM Plex Mono）× 中文衬线（Noto Serif SC）
- **质感**：方格纸（结构色 @ 6-10% 网格线）
- **元件**：做旧圆形印章、尺寸标注线、引线编号①②③、角落定位十字
- **红线种子**：忌纯白纯黑、忌结构色做大色块、忌居中漂浮布局

## 2. 复古印刷 Retro Print

- **气质**：新闻纸、油墨、活字。粗粝、复古、有报刊权威感
- **适合**：评论观点类、历史/文化内容、独立刊物感
- **色板**：底 `#F5F1E8` 新闻纸 / 文字 `#1A1A1A` 油墨黑 / 点缀 `#C0392B` 印刷红 / 次级 `#8B8378` 灰褐
- **字体**：粗壮老式衬线（Bitter / Source Serif 4）× 中文宋体（Noto Serif SC 900）
- **质感**：半调网点、纸张噪点
- **元件**：双细线框（票据感）、首字下沉、报头横线、"第 N 期"编号
- **红线种子**：忌渐变、忌圆角、忌高饱和荧光色

## 3. 杂志编辑 Editorial

- **气质**：大标题、大留白、克制的一抹点缀色。高级、安静、内容为王
- **适合**：深度长文、访谈、生活方式、知识付费
- **色板**：底 `#FFFFFF`→`#FAFAF8` / 文字 `#111111` / 点缀单色（如 `#D4380D` 橘红或 `#0958D9` 克莱因蓝，二选一）/ 次级 `#999999`
- **字体**：超大号现代衬线标题（Libre Caslon / Lora）× 无衬线正文（Noto Sans SC）
- **质感**：无纹理，靠留白和字号层级
- **元件**：极细分隔线（0.5-1px）、页码/栏目角标、引文超大引号
- **红线种子**：忌元素拥挤、忌多于一个点缀色、忌描边字

## 4. 极简瑞士 Swiss Grid

- **气质**：网格、对齐、功能主义。理性、现代、信息密度可以很高
- **适合**：数据/图表向内容、产品测评、效率工具教学
- **色板**：底 `#F4F4F4` / 文字 `#0A0A0A` / 点缀 `#E30613` 信号红 / 辅助 `#B5B5B5`
- **字体**：Grotesk 无衬线全家桶（Archivo / Space Grotesk）× 中文黑体（Noto Sans SC）
- **质感**：无纹理；可见网格线作为设计元素
- **元件**：色块分区、超大序号、箭头符号 →、左对齐一切
- **红线种子**：忌衬线、忌装饰性元素、忌居中对齐

## 5. 暗黑终端 Terminal

- **气质**：近黑底、荧光单色、等宽字。极客、夜间、代码原生
- **适合**：编程教学、命令行工具、黑客向内容（暗色系唯一原型）
- **色板**：底 `#0D1117` / 文字 `#C9D1D9` / 点缀 `#3FB950` 终端绿（或 `#E3B341` 琥珀）/ 结构 `#30363D`
- **字体**：等宽为主（JetBrains Mono / IBM Plex Mono）× 中文黑体（Noto Sans SC）
- **质感**：细扫描线或微噪点（可选）
- **元件**：`$` 提示符、闪烁光标块、ASCII 框线、状态栏
- **红线种子**：忌彩虹语法高亮配色当品牌色、忌亮底、忌衬线

## 6. 手账涂鸦 Notebook

- **气质**：横线本、马克笔、胶带贴纸。亲切、随性、人味儿重
- **适合**：个人 vlog 式分享、学习笔记、生活技巧
- **色板**：底 `#FFFDF5` 奶油 / 文字 `#3D3A35` 深暖灰 / 点缀多彩低饱和（`#F4A261` 橘 `#2A9D8F` 青 `#E76F51` 红）
- **字体**：手写体标题（LXGW WenKai / Ma Shan Zheng）× 圆润正文（Noto Sans SC）
- **质感**：横线纸 / 点阵纸
- **元件**：荧光笔划线、和纸胶带、手绘箭头圈圈、便利贴卡片
- **红线种子**：忌精确对齐（要有手感偏移）、忌企业感图标、忌深色底

## 7. 柔和粉彩 Soft Pastel

- **气质**：低饱和粉彩、大圆角、柔光。温柔、治愈、零攻击性
- **适合**：女性向、亲子、心理/健康、美学生活
- **色板**：底 `#FBF7F4` 奶白 / 文字 `#5C5470` 灰紫 / 点缀 `#F2BAC9` 粉 `#B8D8D8` 雾蓝 `#F7E3AF` 鹅黄
- **字体**：人文无衬线（Quicksand / Varela Round）× 中文圆体气质（Noto Sans SC 300-500）
- **质感**：无纹理或极淡渐变
- **元件**：大圆角卡片（16px+）、柔光阴影、圆点装饰、丝带标签
- **红线种子**：忌高对比黑、忌直角、忌密集排版

## 映射建议

| 访谈信号 | 倾向原型 |
|---------|---------|
| 开发者/技术/工具 | 工程蓝图、极简瑞士、暗黑终端 |
| 观点/评论/文化 | 复古印刷、杂志编辑 |
| 生活/个人成长 | 手账涂鸦、柔和粉彩、杂志编辑 |
| 气质词含"专业/可信" | 工程蓝图、杂志编辑、极简瑞士 |
| 气质词含"亲切/温暖" | 手账涂鸦、柔和粉彩、工程蓝图 |
| 受众主要在深夜刷手机 | 考虑暗黑终端（唯一暗色系） |

混血也合法：用户"工程蓝图的元件 + 粉彩的色板"这种组合诉求，picker 的分区块机制天然支持。
